O sistema de cadastro de clientes solicita o preenchimento do formulário eletrônico.
Para realizar a operação, favor clicar no menu de cadastro.

Para ver os clientes registrados no sistema, clique no menu de lista.

O sistema foi desenvolvido para ser usado diretamente por um funcionário de uma empresa
com fim de salvar os dados de clientes.
